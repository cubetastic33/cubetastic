Info

Welcome to cubetastic timer! This timer has a lot of uses. One of the most striking features of this timer, is that it uses the official TNoodle scrambling program! You can add it to your homescreen like any other app (click the install icon in the menu for more details), and this website also works offline!
When you're online, you get even more features like syncing your times! Once you sign in to your account, any solve you do after that will get synced. Next time, when you are using some other device or browser, you just need to sign in to your account, and all your times on this device will appear there! If you have any feedback, you can click the feedback icon in the menu, and it will redirect you to a feedback page.
This website is also open source, which means you can view the source code! To do this, click the github icon (a cat) in the footer below.
I have also used some other open source libraries, like JQuery and Materialize CSS, which have an MIT licence.

Install the PWA

This website is a PWA, which means you can add it to your homescreen, and it will work just like any other app, even offline! You can install this (if your browser supports it) by clicking the menu of your browser, and then clicking "Add to homescreen". For more detailed browser - specific instructions, click here.

Tested on Chrome (Mobile & Desktop), Firefox (Mobile), Opera (Desktop)
